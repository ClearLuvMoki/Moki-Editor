# Introduction

**Moki Editor** provides a multifunctional editor, written with **<a href="https://tiptap.dev">TipTap</a> + React +
Typescript**;

- 😍 DIY Toolbar/InlineBar
- 💎 Export Json/Html
- ⚙️ Typescript declaration
- 💌 Free and open source
